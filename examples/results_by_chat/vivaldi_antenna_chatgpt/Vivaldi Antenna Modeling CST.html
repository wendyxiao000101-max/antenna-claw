Below is a suggested, orderly procedure for creating the geometry of the Vivaldi antenna in CST. The outline follows the description of each region (substrate, front tapered slot region, and back feeding region), with notes on important dimensions and parameter definitions.

1. Set up the basic CST project

  1. Create a new CST project (e.g., Microwave Studio).
  2. Set the units: Typically, set Length Unit to mm and Frequency Unit to GHz.
  3. Define parameters you will need (suggested as global parameters for easy tuning later if needed). Examples:
    • ts = 0.813 (substrate thickness)
    • tp = 0.035 (copper thickness)
    • l1, w1, l2, w2 (dimensions of back rectangles)
    • pf (the y-offset of the back feed)
    • r1, r2 (circle radii on front and back)
    • gap (the distance used for circle center on the front, if needed)
    • Any other geometrical variables you wish to parametrize (like the spline x-coordinates).

2. Model the substrate

  1. Open the “Modeler” tab, choose Create → Brick (or “Block” in older CST versions).
  2. Set the brick’s corner coordinates so that:
    • X-range: [0,30][0, 30][0,30]
    • Y-range: [0,20][0, 20][0,20]
    • Z-range: [0,ts][0, ts][0,ts] (using ts = 0.813)
  3. Name this solid something like "Substrate".
  4. Assign material to "Rogers RO4003C" (or create/import the material if not already in the library).
  5. Confirm that this substrate block fully covers [0,30][0,30][0,30] in X, [0,20][0,20][0,20] in Y, and [0,ts][0,ts][0,ts] in Z.

You should now have a single rectangular block representing the dielectric substrate.

3. Model the front patch (before subtracting the slot)

    ' 1) Substrate thickness
    names(1) = "ts"
    values(1) = "0.813"
    
    ' 2) Copper thickness
    names(2) = "tp"
    values(2) = "0.035"
    
    ' 3) l1 (length of the first back rectangle)
    names(3) = "l1"
    values(3) = "5"
    
    ' 4) w1 (width of the first back rectangle)
    names(4) = "w1"
    values(4) = "2"
    
    ' 5) l2 (length of the second back rectangle)
    names(5) = "l2"
    values(5) = "8"
    
    ' 6) w2 (width of the second back rectangle)
    names(6) = "w2"
    values(6) = "1"
    
    ' 7) pf (Y-offset for the first back rectangle)
    names(7) = "pf"
    values(7) = "10"
    
    ' 8) gap (distance from X=15 to circle center on front)
    names(8) = "gap"
    values(8) = "0.5"
    
    ' 9) r1 (front circle radius)
    names(9) = "r1"
    values(9) = "0.8"
    
    ' 10) r2 (back circle radius)
    names(10) = "r2"
    values(10) = "0.6"
    
    ' Next 20 parameters = xSpline1 ... xSpline20
    ' Example: linearly spread from 1.0 mm up to ~14.0 mm
    ' (You can revise as needed for your actual slot shape.)
    
    Dim i As Integer
    Dim xVal As Double
    
    For i = 1 To 20
        names(10 + i) = "xSpline" & CStr(i)

        xVal = 1.0 + (14.0 - 1.0) * (i - 1) / 19
        values(10 + i) = CStr(Round(xVal, 3))
    Next i
    
    ' Finally, call the CST built-in macro function to store them
    StoreParameters names, values
    
End Sub

Below is an example .bas file that demonstrates how to create and extrude a closed spline for the tapered slot, and then subtract it from the front patch. We assume:

  • You already have 20 parameters (xSpline1 … xSpline20) for the left spline’s X-coordinates.
  • The Y coordinates descend from 20 down to 1 (i.e., points at (xSpline1, 20), (xSpline2, 19), ..., (xSpline20, 1)).
  • The right spline is symmetrical about x=15x=15x=15. Hence its points are (30−xSpline1,20)(30 - xSpline1, 20)(30−xSpline1,20), (30−xSpline2,19)(30 - xSpline2, 19)(30−xSpline2,19), ..., (30−xSpline20,1)(30 - xSpline20, 1)(30−xSpline20,1).
  • After creating both splines, we connect their top endpoints and bottom endpoints with straight lines, forming a single closed shape (all under one curve name).
  • We then extrude this closed shape with thickness = tp (the metal thickness).
  • Finally, we translate the extruded solid to the correct Z-range [ts,ts+tp][ts, ts + tp][ts,ts+tp] and subtract it from "FrontPatch" to form the open slot.

Note

  1. This sample explicitly lists each .LineTo call (no loops).
  2. The parameter values (ts, tp, xSpline1...xSpline20, etc.) must already be defined in your CST project.
  3. You may rename any items (slotCurve, SlotSolid, etc.) to suit your convention.
vba
' ===========================================================================
' Macro: CreateTaperedSlot.bas
'
' Purpose:
'   1. Build a closed spline (two 20-point splines + top/bottom lines).
'   2. Extrude the closed curve into a 3D solid.
'   3. Translate the solid to [ts, ts+tp] in Z.
'   4. Subtract from "FrontPatch" to form the tapered slot.
'
' Prerequisites:
'   - "FrontPatch" (Copper) already exists in "component1".
'   - Parameters:
'       ts = Substrate thickness
'       tp = Copper thickness
'       xSpline1...xSpline20  (Left spline X-coords)
'     in CST's Parameter List.
'
' ===========================================================================
Option Explicit   ' (Optional; can be removed if you prefer.)

Sub CreateTaperedSlot()

    ' --- 1) Create the LEFT spline (20 points) ---
    '     Curve name: "slotCurve"
    '     Spline name: "LeftSpline"
    With Spline
        .Reset
        .Name "LeftSpline"
        .Curve "slotCurve"
        
        ' First point:
        .Point "xSpline1", "20"  
        .SetInterpolationType "PointInterpolation"
        
        ' 19 subsequent lines (Y descends from 19 down to 1)
        .LineTo "xSpline2",  "19"
        .LineTo "xSpline3",  "18"
        .LineTo "xSpline4",  "17"
        .LineTo "xSpline5",  "16"
        .LineTo "xSpline6",  "15"
        .LineTo "xSpline7",  "14"
        .LineTo "xSpline8",  "13"
        .LineTo "xSpline9",  "12"
        .LineTo "xSpline10", "11"
        .LineTo "xSpline11", "10"
        .LineTo "xSpline12",  "9"
        .LineTo "xSpline13",  "8"
        .LineTo "xSpline14",  "7"
        .LineTo "xSpline15",  "6"
        .LineTo "xSpline16",  "5"
        .LineTo "xSpline17",  "4"
        .LineTo "xSpline18",  "3"
        .LineTo "xSpline19",  "2"
        .LineTo "xSpline20",  "1"
        
        .Create
    End With

    ' --- 2) Create the RIGHT spline (symmetric about x=15) ---
    '     Spline name: "RightSpline"
    With Spline
        .Reset
        .Name "RightSpline"
        .Curve "slotCurve"

        ' First point: (30 - xSpline1, 20)
        .Point "30 - xSpline1", "20"
        .SetInterpolationType "PointInterpolation"

        ' Next 19 lines: (30 - xSpline2, 19) ... (30 - xSpline20, 1)
        .LineTo "30 - xSpline2",  "19"
        .LineTo "30 - xSpline3",  "18"
        .LineTo "30 - xSpline4",  "17"
        .LineTo "30 - xSpline5",  "16"
        .LineTo "30 - xSpline6",  "15"
        .LineTo "30 - xSpline7",  "14"
        .LineTo "30 - xSpline8",  "13"
        .LineTo "30 - xSpline9",  "12"
        .LineTo "30 - xSpline10", "11"
        .LineTo "30 - xSpline11", "10"
        .LineTo "30 - xSpline12",  "9"
        .LineTo "30 - xSpline13",  "8"
        .LineTo "30 - xSpline14",  "7"
        .LineTo "30 - xSpline15",  "6"
        .LineTo "30 - xSpline16",  "5"
        .LineTo "30 - xSpline17",  "4"
        .LineTo "30 - xSpline18",  "3"
        .LineTo "30 - xSpline19",  "2"
        .LineTo "30 - xSpline20",  "1"

        .Create
    End With

    ' --- 3) Connect top endpoints with a straight line ---
    '     From (xSpline1, 20) to (30 - xSpline1, 20)
    With Line
        .Reset
        .Name "TopLine"
        .Curve "slotCurve"
        .X1 "xSpline1"         ' left top endpoint
        .Y1 "20"
        .X2 "30 - xSpline1"    ' right top endpoint
        .Y2 "20"
        .Create
    End With

    ' --- 4) Connect bottom endpoints with a straight line ---
    '     From (xSpline20, 1) to (30 - xSpline20, 1)
    With Line
        .Reset
        .Name "BottomLine"
        .Curve "slotCurve"
        .X1 "xSpline20"
        .Y1 "1"
        .X2 "30 - xSpline20"
        .Y2 "1"
        .Create
    End With

    ' At this stage, "slotCurve" is a CLOSED shape:
    '   - LeftSpline + RightSpline + TopLine + BottomLine
    '   - We'll extrude this shape to form the copper volume to subtract.

    ' --- 5) Extrude the closed curve (thickness = tp) ---
    '     By default, it starts from Z=0 to Z=tp
    With ExtrudeCurve
        .Reset
        .Name "SlotSolid"
        .Component "component1"
        .Material "Copper (pure)"
        .Thickness "tp"
        .Twistangle "0.0"
        .Taperangle "0.0"
        
        ' If you want to remove the 2D curve after extruding, set .DeleteProfile = "True"
        .DeleteProfile "False"
        
        .Curve "slotCurve"
        .Create
    End With

    ' --- 6) Translate the extruded solid to [ts, ts + tp] in Z ---
    '     Because it’s currently at [0, tp].
    With Transform
        .Reset
        .Name "component1:SlotSolid"    ' or "component1:SlotSolid:1" depending on CST version
        .Vector "0", "0", "ts"         ' shift upward by ts
        .UsePickedPoints "False"
        .InvertPickedPoints "False"
        .MultipleObjects "False"
        .GroupObjects "False"
        .Repetitions "1"
        .MultipleSelection "False"
        .Destination ""
        .Material ""
        .AutoDestination "True"
        .Transform "Shape", "Translate"
    End With

    ' --- 7) Subtract from "FrontPatch" to form the tapered slot ---
    Subtract "component1:SlotSolid", "component1:FrontPatch"

End Sub

Explanation of Key Steps

  1. Left Spline: We define 20 points from top to bottom:
    • xSpline1, 20 → xSpline2, 19 → … → xSpline20, 1
  2. Right Spline: Mirror the same points around x=15x=15x=15, i.e. 30 - xSplineN.
  3. Top/Bottom Lines: Closes the shape at y=20y=20y=20 (top) and y=1y=1y=1 (bottom).
  4. Extrude: Creates a 3D copper solid from Z=0 to Z=tp with the name "SlotSolid".
  5. Translate: Moves "SlotSolid" in the Z-direction by ts, so its final vertical range is [ts, ts + tp].
  6. Subtract: Removes this 3D slot from the "FrontPatch", leaving the tapered opening.

This completes the creation of the tapered slot in the front copper region.
